| 方法 | 计算量推导 | 参数量推导（分步） |
|------|------------|--------------------|
| **Ensemble of GCs** | 1. 单个GCN层计算量：$\quad \vert\mathcal{V}\vert^2 d + \vert\mathcal{V}\vert d^2$ | 1. 单个GCN层参数量：$\quad \vert\mathcal{V}\vert^2 + d^2$ |
| | 2. K个独立层总计算量：$\quad K(\vert\mathcal{V}\vert^2 d + \vert\mathcal{V}\vert d^2)$ | 2. K个独立层总参数量：$\quad K(\vert\mathcal{V}\vert^2 + d^2)$ |
| | 3. 当 $d \gg \vert\mathcal{V}\vert$ 时：$\quad \approx \boldsymbol{\mathcal{O}(K \vert\mathcal{V}\vert d^2)}$ | 3. 展开后：$\quad = \boldsymbol{K \vert\mathcal{V}\vert^2 + K d^2}$ |
| **Proposed BlockGC** | 1. 单个分组计算量：$\quad \vert\mathcal{V}\vert^2 \frac{d}{K} + \vert\mathcal{V}\vert \left(\frac{d}{K}\right)^2$ | 1. 邻接矩阵参数量：$\quad K \vert\mathcal{V}\vert^2$（与Ensemble完全相同） |
| | 2. K个分组总计算量：$\quad \vert\mathcal{V}\vert^2 d + \frac{\vert\mathcal{V}\vert d^2}{K}$ | 2. 分块权重总参数量：$\quad \sum_{k=1}^K \left(\frac{d}{K}\right)^2 = \frac{d^2}{K}$ |
| | 3. 当 $d \gg \vert\mathcal{V}\vert$ 时：$\quad \approx \boldsymbol{\mathcal{O}\left( \frac{\vert\mathcal{V}\vert d^2}{K} \right)}$ | 3. 总参数量：$\quad = \boldsymbol{K \vert\mathcal{V}\vert^2 + \frac{d^2}{K}}$ |

> **📌 计算量推导基础公式**
> 两个矩阵相乘 $Z = X \times Y$，其中 $X \in \mathbb{R}^{M \times N}$，$Y \in \mathbb{R}^{N \times P}$，则总浮点运算量为：
> $$ \boldsymbol{\text{FLOPs} = 2 \times M \times N \times P} $$
> GCN的计算量由两次矩阵乘法组成：邻接矩阵与特征相乘（$A \times H$）+ 结果与权重矩阵相乘（$(AH) \times W$）
> 
> **符号说明**：$\vert\mathcal{V}\vert$ = 人体关节数，$d$ = 特征维度，$K$ = 多关系数量
> 
> **核心前提**：骨骼数据天然满足 $\boldsymbol{d \gg \vert\mathcal{V}\vert}$（通常 $d=256, \vert\mathcal{V}\vert=25$），因此 $\vert\mathcal{V}\vert^2 d$ 项可忽略，权重矩阵乘法的复杂度占绝对主导。
